This will add additional feature to allow your customer to put delivery date and address with some gift message of their own choice per product.

== Description ==

Shipping & Delivery Date management with gift message extension allows selection of delivery date, messaging of gift hampers, shipping address management feature. All these features are available on product page. By using this extension the admin can gives various functionalities to the customers like:

<strong>For additional features of Shipping & Delivery Date management with gift message, upgrade to the premium version of <a href="https://cedcommerce.com/woocommerce-extensions/shipping-delivery-date-management-with-gift-message/">Shipping & Delivery Date management with gift message</a></strong>.


* Gift message for every product can be set by customer.
* On the basis of customer's choice, his delivery date would be selected.
* Shipping address for every product on product page can be set by customer.

= Customer can set = 

* **Recipient address-** To whom he wants to send the product. 
* **Delivery Date-** The date he wants to receive the product.
* **Delivery Time-** The time range between which he wants to receive the product. 
* **Gift Message-** Text message can be set that he/she wants to send with the product on every product page. This feature is for both logged in and non-logged in customers.

The Admin can also customize this extension like he can enable or disable the extension, can customize Delivery date like week off, nearest order delivery date, disable Past day, can add extra cost based on days, can disable multiple dates, can choose multiple days he is not able to supply orders and can also disable /enable specific feature of this extension.

= Main Features of Free Version  =

* Prevent delivery on selected days.
* Option to deliver on all week days.
* Add extra cost based on days.
* Enable / Disable delivery date feature on frontend.
* Enable / Disable gift message feature on frontend.
* Enable / Disable shipping address feature on frontend.
* Gift message for every product can be set by customer.
* On the basis of customer's choice, his delivery date would be selected.
* Shipping address for every product on product page can be set by customer.
* Order email send with product additional information
* Show-hide delivery address at cart/checkout/order pages to win space.
* Adjust the spacing/padding around the position of the gift box.
* Option for displaying shipping and delivery form on selected product categories.
